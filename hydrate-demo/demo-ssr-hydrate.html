SSR (服务端渲染) 与客户端水合演示

本演示直观展示了服务端渲染页面后，为什么仍需要将数据传递到客户端并执行水合过程。

请打开浏览器控制台 (F12) 查看详细的执行过程。

1. 服务端渲染的内容

这部分HTML是由服务器预先渲染的，包含了用户可见的内容。

2. 服务端传递的数据

服务器通过 <script> 标签将数据序列化并挂载到 window 对象上。

window.__SERVER_DATA__ = {

"products": [

{"id": 1, "name": "产品A", "price": 100},

{"id": 2, "name": "产品B", "price": 200},

{"id": 3, "name": "产品C", "price": 300}

],

"pageInfo": {"title": "产品列表", "renderTime": "2023-11-09T10:00:00Z"}

};
